== Description ==

Flip your website upside down with CSS transforms.

== Installation ==

1. Upload `flip.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Ensure that your template uses `wp_head()`
1. Load a page with the GET variable, `flip=1`
